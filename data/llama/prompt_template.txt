For the following assay_description, extract the following structured metadata fields according to the logic below. All outputs must be lowercase strings unless otherwise stated.

---

Fields & Instructions:

mtb_strain:
- If the text mentions a specific strain of *Mycobacterium tuberculosis*, return the full strain reference as it appears, preserving both strain names and catalog identifiers when combined.
  - Examples:
    - "H37Rv ATCC 27294" → return "H37Rv ATCC 27294"
    - "Mycobacterium tuberculosis H37Rv isolate BK12" → return "H37Rv isolate BK12"
    - "Mycobacterium tuberculosis ATCC 35837" → return "ATCC 35837"
    - "H37Rv" → return "H37Rv"
- If only the species name *Mycobacterium tuberculosis* is mentioned, return: "mycobacterium tuberculosis"
- If nothing related to strain or species is found, return: false

mentions_resistance:
- If the text does not mention resistance, return: false
- If it mentions extensively drug-resistant, return: "xdr"
- If it mentions multi-drug-resistant or multidrug-resistant, return: "mdr"
- If it mentions both resistant/extensively drug resistant, return: "mdr/xdr"
- If it mentions any other form of resistance (e.g., "isoniazid-resistant", "resistant to rifampin"), return: "R"

resistant_to:
- If the assay description mentions resistance to specific drug or drugs, return the drug names in lowercase, comma-separated (e.g., "rifampin, isoniazid")
- If multidrug resistance is mentioned (e.g., "multidrug-resistant") but no specific drugs are listed, return: "MDR"
- If resistance is not mentioned, return: false

mutant:
- If a mutant strain or engineered variant is mentioned (e.g., with plasmid, gene overexpression, or point mutation), return: true
- Otherwise, return: false

mutant_type:
- If a mutation or construct is specified (e.g., "katG S315T", "pMV261::InhA", "pFCA-lux AB"), return it as-is
- If not found, return: false

checkerboard:
- If the text mentions a checkerboard assay, return: true
- Otherwise, return: false

---

All boolean values (`true`/`false`) must be lowercase. Strings must also be lowercase.
